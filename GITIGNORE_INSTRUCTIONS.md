# Инструкции по настройке .gitignore

Для обеспечения безопасности конфиденциальных данных, пожалуйста, добавьте следующие строки в ваш файл .gitignore:

```
# Конфигурационные файлы с секретами
config_secrets.json

# Виртуальное окружение Python
venv/
.env

# Байт-скомпилированные файлы Python
__pycache__/
*.py[cod]
*$py.class

# Дистрибутивы и пакеты
*.egg-info/
dist/
build/
```

Это предотвратит попадание конфиденциальных данных в репозиторий Git.
